# Atividade 1 - Site estilo globo.com com TABLE

Projeto simples em **HTML + CSS** para a atividade:

> "Montar o site da globo.com utilizando tabelas (`<table>`) e publicar no GitHub Pages."

O layout principal foi construído com tabelas (`table`, `tr`, `td`, `th`) e mantém o estilo de um portal de notícias.

## Estrutura de pastas

```text
.
├── index.html
├── style.css
└── README.md
```

## Como rodar localmente

1. Baixe/clone o repositório.
2. Abra o arquivo `index.html` diretamente no navegador.
3. Pronto: a página será carregada sem necessidade de servidor.

## Como publicar no GitHub Pages

1. Acesse o repositório no GitHub.
2. Clique em **Settings**.
3. No menu lateral, clique em **Pages**.
4. Em **Build and deployment**, escolha:
   - **Source**: `Deploy from a branch`
   - **Branch**: `main`
   - **Folder**: `/ (root)`
5. Clique em **Save**.
6. Aguarde alguns instantes e abra o link gerado do GitHub Pages.
